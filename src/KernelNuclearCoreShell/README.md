# Kernel Nuclear Core and Shell/核核核殻
Copyright (C) 2026 Takym.

## 概要
オペレーティングシステムの開発を練習する為のものです。
名称にカーネルの類義語が列記されている通り当分はカーネル寄りの開発になると思います。

## 依存関係
* Windows 10 以上
* WSL 有効化必須
	* Ubuntu-24.04 のインストール必須

### ブートローダー
* [NASM](https://www.nasm.us)
* [QEMU](https://www.qemu.org/)
